# Decimus Precision static export

Static files extracted from the provided MHTML archive and prepared for GitHub Pages.

## Structure

- `index.html` — static landing page
- `css/styles.css` — extracted site styles plus small static-export helpers
- `css/fonts.css` — extracted Google Fonts stylesheet from the MHTML archive
- `css/base44-badge.css` — extracted badge animation CSS from the archive
- `js/main.js` — small helper script for mobile menu, sticky header state, and badge close button
- `assets/img/` — extracted images and inline image assets
- `manifest.json` — minimal local manifest replacement

## GitHub Pages

Upload these files to the root of the repository, then enable GitHub Pages for the branch/folder you use.
